## 技术栈

所有文章基于以下技术栈：

- **语言**：Python 3.10+
- **大模型**：OpenAI GPT-4o-mini（可替换为 DeepSeek / 通义千问等）
- **向量库**：ChromaDB（入门）/ FAISS / Qdrant（进阶）
- **框架**：LangChain / LlamaIndex / LangGraph / CrewAI / AutoGen（按需）
- **其他**：NetworkX（图谱）、RAGAS（评估）、MCP SDK（工具接入）
